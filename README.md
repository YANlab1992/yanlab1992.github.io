# 小智的数智乐园 - 心理陪伴系统

一个专业的心理学研究辅助工具平台，提供智能对话、心理测评、日记记录和心理信件等功能。

## 🌟 功能特点

### 1. 智能对话模块 💬
- 与AI心理陪伴助手对话
- 支持对话历史记录
- 最多保存10轮对话上下文
- 温暖的情感支持回复

### 2. 心理测评模块 📋
- **PHQ-9 抑郁筛查量表** - 评估抑郁症状
- **GAD-7 焦虑筛查量表** - 评估焦虑症状
- **SWLS 主观幸福感量表** - 评估生活满意度
- **PANAS 正负性情感量表** - 评估积极/消极情感
- **CD-RISC 心理弹性量表** - 评估心理韧性
- **简版共情量表** - 评估共情能力
- **简版睡眠质量量表** - 评估睡眠状况
- **知觉压力量表** - 评估压力感知

### 3. 心理日记模块 📔
- 每日心情记录
- 8种情绪选择（开心、兴奋、平静、疲惫、焦虑、难过、生气、迷茫）
- 相关话题标签
- 日历视图查看历史
- 数据导出功能

### 4. 心理信件模块 💌
- 给未来自己/过去自己/朋友/家人/爱人的信
- 定时开启功能（可设置未来日期）
- AI辅助写作建议
- 隐私保护（仅本地存储）

## 🎨 设计规范

### 配色方案
- **主色调**: 温暖的蓝绿色系 (#2E8B8B)
- **辅助色**: 柔和的暖色调
- **情绪颜色**: 对应不同情绪的专属色彩

### 界面风格
- 卡片式布局
- 圆角设计
- 温馨、专业的心理学期刊风格
- 响应式设计，支持手机端

## 🚀 使用说明

### 直接使用
1. 下载所有文件到本地目录
2. 使用现代浏览器打开 `index.html` 即可运行
3. 无需服务器，所有数据存储在浏览器本地 (LocalStorage)

### 与现有系统集成
- 对话功能默认连接: `https://yanzqvip.pythonanywhere.com/chat`
- 可根据需要修改 `modules/chat.js` 中的 `apiEndpoint`

## ⚠️ 免责声明

本平台提供的心理测评和对话功能**仅供自评参考**，不构成临床诊断或治疗建议。如用户正在经历严重的心理困扰，请及时寻求专业心理工作者或精神科医生的帮助。

## 🔒 隐私说明

- 所有数据仅存储在用户本地设备
- 不会上传到任何服务器
- 用户可随时导出或清除数据
